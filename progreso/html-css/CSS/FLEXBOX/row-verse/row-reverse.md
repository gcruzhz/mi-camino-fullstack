# 
ROW-REVERSE

Para Cambiar la posicion inicial de los elementos flexibles al lado derecho de un contenedor, podemos usar ROW-REVERSE

flex-direction: row-reverse significa que los elementos
comienzan desde la derecha del contenedor y se muestran en el orden que estan en el HTML

¿Que hace row-reverse?
Invierte la posicion inicial de los elementos flexibles mostrados en una fila

¿que mostrara el codigo html?
Los numeros 4, 3, 2, 1 en el lado derecho del contenedor
